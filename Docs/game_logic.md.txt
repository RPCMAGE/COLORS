# COLORS ‚Äî Game Logic

## Overview
COLORS is a multi-color betting game inspired by the Filipino ‚ÄúPerya‚Äù color game.  
Players bet on up to three colors, and three colored dice are rolled.  
Winnings depend on how many dice show the colors you bet on.

The game is fast, simple, and designed to give satisfying rewards even on small wins.

---

## üé≤ Dice & Colors
- 3 large dice, each with 6 sides: **Yellow, Orange, Pink, Blue, Green, Red**.  
- Players can bet on **up to 3 colors per round**.
- Each color bet is independent (e.g. 100 on Green, 50 on Red).

---

## ‚öôÔ∏è Game Modes

### Normal Mode
Slower, standard mode ‚Äî players have time to bet before the roll.  
Rewards are moderate and balanced.

| Matches | Added Multiplier | Example Return (100 bet) | Notes |
|----------|------------------|---------------------------|--------|
| 1 color match | +0.45√ó | 145 | Small win |
| 2 same colors | +0.95√ó | 195 | Moderate win |
| 3 colors (mixed match) | +1.69√ó | 269 | Strong win |
| 3 of the same color (jackpot) | **+2.25√ó** | **325** | Rare big win |

> üîπ Multipliers are **additive** ‚Äî you keep your bet + earn the multiplier on top.  
> So even a small win feels rewarding rather than a net loss.

---

### Time Attack Mode
Fast-paced and high-risk.  
Players get **5‚Äì10 seconds** to place bets.  
If no bet is placed in time ‚Üí skip 2 rounds.  
Rewards are slightly higher to match the urgency.

| Matches | Added Multiplier | Example Return (100 bet) |
|----------|------------------|---------------------------|
| 1 color match | +0.55√ó | 155 |
| 2 same colors | +1.05√ó | 205 |
| 3 colors (mixed match) | +1.95√ó | 295 |
| 3 of the same color (jackpot) | **+2.55√ó** | **355** |

## üïπÔ∏è Game Flow
1. Player selects bet amount and 1‚Äì3 colors.  
2. Press **Roll** to start.  
3. Dice animate (~1.5 seconds).  
4. Results displayed ‚Üí winnings calculated.  
5. Balance updates automatically.  
6. New round starts.

## üßÆ Implementation Example
```js
function calculatePayout(bet, matches, mode, allSameColor = false) {
  const multipliers = {
    normal: { 1: 0.45, 2: 0.95, 3: 1.69 },
    attack: { 1: 0.55, 2: 1.05, 3: 1.95 }
  };
  
  let added = multipliers[mode][matches] || 0;
  if (allSameColor && matches === 3) {
    added += (mode === "attack") ? 0.6 : 0.56; // jackpot bonus bump
  }
  return bet * (1 + added);
}
